Zijnalle properties die de generator voorziet nodig?

Nee. Zo voorziet Table Style Generator ook selectoren voor bv. links en lijsten, wat in ons voorbeeld niet van toepassing is.
Deze kunnen dus weggelaten worden. Er is standaard ook een footer voor de tabel. Deze is ook niet van toepassing voor ons.

De generator gebruikt voor bepaalde properties een browser prefix. Zijn die voor jouw browser nog nodig?

Deze prefixes worden gebruikt om oudere browsers te kunnen ondersteunen, en nieuwe toepassingen van HTML en CSS toch te kunnen gebruiken
Voor meeste mensen zijn deze overbodig, zolang je een recente en odersteunde browser gebruikt, zoals Chrome, Firefox, ect.

Opdracht 3

In CSS worden de elementen met de klasse .alt als volgt gedefinieerd: "table tbody .alt"
Deze notatie is een stuk specifieker dan simpelweg de klasse .onbekend gebruiken, omdat het ook nog tbody aanspreekt.
De specificiteit is dus groter dan enkel .onbekend gebruiken.
